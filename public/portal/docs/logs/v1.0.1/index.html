QAdmin-pjax v1.0.1 更新日志

2017/06/13

QAdmin-pjax v1.0.1是一个维护版本，重点以优化和修复bug为主，v1.0.1 更新日志如下：

优化

  • 优化核心js文件，将个别方法进行迁移；
  • 优化window.Content对象，为其增加扩展功能。解决一个主体内容部分是由两个引用了包含window.Content对象js文件的页面组成时，导致引用只指向第二个对象的问题；
  • 优化nodemon的配置选项。在开发阶段，精准监听服务端文件。忽略不必要的文件监听；
  • pjax逻辑优化。解决项目中不同路径写法导致错误获取问题；
  • 优化通知消息数目。当消息数大于99条时显示为99+；
  • 优化个别页面的书写规范。

修复

  • 修复菜单管理页面移动菜单导致的相关菜单信息出现的变化错误问题；
  • 修复前端源码项目中网站地图路径错误问题；
  • 修复显示设置中tab页签设置按钮传值错误问题；
  • 修复显示设置中tab页签在默认关闭时，重新开启页签时的错误问题；
  • 修复表格树大量数据无法加载问题；
  • 修复前端源码项目中忽略掉的关于错误页面不同加载方式的处理。
