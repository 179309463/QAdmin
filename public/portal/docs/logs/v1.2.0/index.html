QAdmin-pjax v1.2.0 更新日志

2017/09/12

QAdmin-pjax v1.2.0 重点改进了gulp任务以及核心样式文件，同时优化和修复若干问题，v1.2.0 更新日志如下：

改进

  • 改进gulp构建任务相关配置，提高运行效率，优化处理方式；

优化

  • 优化主题显示效果；
  • 优化页面示例-团队协作-知识库部分页面；
  • 删除标签页相关无用代码；
  • 删除页面中一些无用的文件；

修复

  • 修复菜单管理、用户管理等页面无法使用公用方法（app.js文件)问题；
  • 修复表格示例-dataTables-基本初始化-垂直滚动条页面表格底部错误问题；
  • 修复显示设置中主题设置为其他颜色后，导航条颜色不能重新设置成蓝色（默认色）问题；
  • 修复气泡提示&弹出框页面iframe按钮地址配置问题；
  • 修复动画面板、动画操作页面通过pjax加载后js执行异常问题；
  • 修复登录页面验证码图片地址错误问题；
  • 修复使用$.pjax方法是没有配置replace参数导致的异常；
