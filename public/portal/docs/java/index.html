概述

QAdmin 使用了JDK1.7.0_05作为JAVA版后台开发的JDK，MAVEN来管理项目的构建，数据库使用的是MySQL5.5.20，我们会陆续推出其他数据库（如:Oracle）的版本，请及时关注。

环境支持

QAdmin 可以很好的支持JDK1.7及以上版本，MySQL5.5及以上版本，操作系统不限。

开源框架

QAdmin 使用了Spring MVC + MyBatis作为基础开发框架：

  • Spring 4.1.6.RELEASE
  • MyBatis 3.2.5

相信很多做过JAVA WEB开发的小伙伴对这套框架已经很熟悉了，即使之前没有接触过，那也完全没有关系。一方面，网上关于此套框架的文档很多；另一方面，在下面的文档中，我们会详细介绍如何将QAdmin 在本地开发工具中导入并正常运行，有了源码和能够正常访问的系统，一切问题都简化了。
