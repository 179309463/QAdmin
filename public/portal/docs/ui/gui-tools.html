开发工具

简介

为了能使开发者快速的测试和部署 QAdmin，我们提供了用于 QAdmin 项目的 GUI 开发工具，包括了 Windows 和 macOS 版本，您可以使用它来快速开发基于 QAdmin 的项目，它可以通过 GUI 界面处理QAdmin的服务开启/关闭，项目部署，并提供了编译提醒，出错提醒，进程关闭提醒等额外的功能。

GUI 开发工具用于自动构建项目，所以仅适用于购买了”前端源码-自动构建”部分的用户。

GUI 开发工具能满足您日常的开发需求，但是一些操作可能仍需要借助命令行来实现，具体请参考：构建工具。
